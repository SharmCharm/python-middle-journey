# 12.05.2026 — Mutability

Сегодня прошёл тему ссылок и mutability в Python.

Главное:
- Переменная в Python это имя, привязанное к объекту, не "коробка со значением"
- `is` проверяет идентичность (один ли объект), `==` — равенство (одинаковое ли содержимое)
- Mutable объекты (list, dict, set) изменяются "на месте"
- `.copy()` это shallow copy — копирует только верхний уровень
- Для глубокой копии нужен `copy.deepcopy()`
- Slice assignment `x[i:j] = [...]` может менять длину списка
